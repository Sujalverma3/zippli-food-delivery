# Zippli - Premium Food Delivery Web App

Inspired by Swiggy. Built with MERN Stack.

## Running the Project
1. Make sure MongoDB is running locally at `mongodb://localhost:27017/zippli`
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start development servers (frontend + backend):
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
